# Tulip

Il gira-chiavi portachiavi.


## Detailed Description

Il gira-chiavi portachiavi.

Panoramica del Progetto

In collaborazione con ABAR, Thinkalize e PACO Design, ci hanno richiesto un gira-chiavi per persone con artrite reumatoide. I classici strumenti, o spezzano le chiavi, o sono discriminanti/ghettizzanti. Insieme alle tue chiavi, hai un 'fiore' e una 'foglia' che diventano una 'maniglia' per girare le chiavi. 

Guida Passo-Passo

1. Scarica i file e stampali in 3D

2. Collega i due componenti con una catenella o altro che possa tenerli uniti
