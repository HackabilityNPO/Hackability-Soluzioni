# Massaggiatore plantare

Massaggiatore plantare per la riattivazione della microcircolazione sanguigna della pianta del piede.


## Detailed Description

Massaggiatore plantare per la riattivazione della microcircolazione sanguigna della pianta del piede.

Panoramica del Progetto

Progetto sviluppato a causa della rottura del tendine di Achille, successivo intervento chirurgico e fase riabilitativa. Il massaggiatore plantare partecipa alla riattivazione della microcircolazione sanguigna. Modellazione 3D mediante Software Fusion 360 e successiva stampa 3D.
